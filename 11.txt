# 🔌 Plugin Manifest System  
*(Declarative, extensible, hot‑swappable)*

A manifest is a small metadata file that sits next to each plugin and describes:

- plugin name  
- version  
- author  
- description  
- category  
- dependencies  
- enabled/disabled state  
- weight (for similarity plugins)  
- permissions (optional)  

This allows your system to load plugins **intelligently**, not blindly.

# ⚙️ 4. Manifest Loader

This loader reads the manifest file **before** loading the plugin class.

```python
# core/manifest_loader.py
import json
import os

def load_manifest(plugin_path: str):
    base = plugin_path.replace(".py", "")
    manifest_path = base + ".manifest.json"

    if not os.path.exists(manifest_path):
        return None

    with open(manifest_path, "r") as f:
        return json.load(f)
```

# 🔍 5. Integrating Manifests Into the Dynamic Plugin Loader

Modify your loader so it:

1. Finds plugin file  
2. Loads manifest  
3. Skips plugin if `enabled: false`  
4. Instantiates plugin class  
5. Attaches manifest metadata to the plugin instance  

```python
# core/plugin_loader.py
import importlib
import pkgutil
import inspect
from plugins.base import SitePlugin, SimilarityPlugin
from core.manifest_loader import load_manifest

def load_plugins(package_path: str, base_class):
    plugins = []
    package = importlib.import_module(package_path)

    for _, module_name, _ in pkgutil.iter_modules(package.__path__):
        module_path = f"{package_path}.{module_name}"
        module = importlib.import_module(module_path)

        manifest = load_manifest(module.__file__)
        if manifest and not manifest.get("enabled", True):
            continue  # skip disabled plugin

        for _, obj in inspect.getmembers(module, inspect.isclass):
            if issubclass(obj, base_class) and obj is not base_class:
                instance = obj()
                instance.manifest = manifest or {}
                plugins.append(instance)

    return plugins
```

# 🔥 7. Hot‑Reloading Manifests

You can reload manifests without restarting the system:

```python
def reload_manifests(plugins):
    for plugin in plugins:
        manifest = load_manifest(plugin.__class__.__module__)
        if manifest:
            plugin.manifest = manifest
```

This allows:

- enabling/disabling plugins live  
- adjusting weights  
- updating descriptions  
- changing permissions
