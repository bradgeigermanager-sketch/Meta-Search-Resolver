# 🔌 Dynamic Plugin Loader (Full System)

The loader will:

1. Scan a directory (`plugins/`)  
2. Import every `.py` file  
3. Find classes that inherit from `SimilarityPlugin` or `SitePlugin`  
4. Instantiate them  
5. Register them automatically  

This gives you a **zero‑touch plugin system**.

    def extract(self, html: str) -> list[dict]:
        ...


class SimilarityPlugin(ABC):
    name: str

    @abstractmethod
    def compare(self, a, b) -> dict:
        ...

# ⚙️ 3. Dynamic Plugin Loader (Core Logic)

This loader:

- walks directories  
- imports modules dynamically  
- finds subclasses of your plugin base classes  
- instantiates them  
- registers them  

```python
# core/plugin_loader.py
import importlib
import pkgutil
import inspect
from plugins.base import SitePlugin, SimilarityPlugin

def load_plugins(package_path: str, base_class):
    plugins = []

    package = importlib.import_module(package_path)

    for _, module_name, _ in pkgutil.iter_modules(package.__path__):
        module = importlib.import_module(f"{package_path}.{module_name}")

        for name, obj in inspect.getmembers(module, inspect.isclass):
            if issubclass(obj, base_class) and obj is not base_class:
                plugins.append(obj())

    return plugins


def load_all_plugins():
    site_plugins = load_plugins("plugins.sites", SitePlugin)
    similarity_plugins = load_plugins("plugins.similarity", SimilarityPlugin)

    return {
        "sites": site_plugins,
        "similarity": similarity_plugins
    }
```

# 🔥 5. Hot‑Swapping Plugins (Optional)

You can reload plugins at runtime:

```python
def reload_plugins():
    global SITE_PLUGINS, SIMILARITY_PLUGINS
    plugins = load_all_plugins()
    SITE_PLUGINS = plugins["sites"]
    SIMILARITY_PLUGINS = plugins["similarity"]
```

This allows:

- adding new plugins  
- updating existing plugins  
- disabling plugins  

**without restarting the system**.

# 🧬 7. Example: Adding a New Plugin (Zero Code Changes)

Drop this file into `plugins/similarity/`:

```python
# plugins/similarity/location_match.py
from plugins.base import SimilarityPlugin

class LocationMatchPlugin(SimilarityPlugin):
    name = "location_match"

    def compare(self, a, b):
        if not a.location or not b.location:
            return {"score": 0, "signals": {}}

        score = 1.0 if a.location.lower() == b.location.lower() else 0.0

        return {
            "score": score,
            "signals": {"location_a": a.location, "location_b": b.location}
        }
```

Restart nothing.  
Touch nothing.  
The system loads it automatically.
