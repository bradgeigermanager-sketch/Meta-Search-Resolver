## 🧩 Step 1 — Normalize URLs from all plugins
Every site plugin should output **canonical URLs**.

Example normalization function:

```python
def normalize_url(url: str) -> str:
    url = url.strip()
    url = url.replace("http://", "https://")
    if url.endswith("/"):
        url = url[:-1]
    return url
```

Every plugin runs:

```python
profile_url = normalize_url(profile_url)
```

## 🧩 Step 3 — Build the Identity URL Registry

After clustering accounts into an identity:

```python
def build_identity_url_registry(identity_id, accounts, artifacts, sources):
    registry = {
        "identity_id": identity_id,
        "urls": {
            "accounts": {},
            "artifacts": {},
            "sources": {},
            "evidence": {}
        }
    }

    # Accounts
    for acc in accounts:
        registry["urls"]["accounts"][acc.source] = acc.profile_url
        for u in acc.urls:
            registry["urls"]["evidence"].setdefault("profile_links", []).append(u)

    # Artifacts
    for art in artifacts:
        kind = art["attributes"]["kind"]
        registry["urls"]["artifacts"].setdefault(kind, []).append(art["id"])

    # Sources
    for src in sources:
        registry["urls"]["sources"].setdefault(src["engine"], []).append(src["url"])

    return registry
```
